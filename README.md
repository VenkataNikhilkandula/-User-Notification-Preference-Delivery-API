# -User-Notification-Preference-Delivery-API# 📢 User Notification Preference & Delivery API

A Spring Boot REST API that manages users and their notification preferences, and delivers notifications based on user settings.

---

## 🚀 Features

### 👤 User Management
- Create a user (name, email, phone)
- Get user by ID
- List all users

### 🔔 Notification Preferences
- Enable/disable notifications
- Choose notification types (Email, SMS, Push)

### 📬 Notification Delivery
- Send notifications based on user preferences
- Supports multiple delivery channels

---

## 🛠️ Tech Stack

- Java 17+
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman (for testing)
